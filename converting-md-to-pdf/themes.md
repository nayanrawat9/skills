# Theme Reference

This document describes all available themes for markdown to PDF conversion.

## Available Themes

**File**: `themes/anthropic-brand.css`

A professional corporate theme following Anthropic's brand guidelines.

| Element | Style |
|---------|-------|
| **Primary** | Dark `#141413` |
| **Background** | Cream `#faf9f5` |
| **Accent** | Orange `#d97757` |

## Code Highlighting Styles

Available `--highlight-style` options:
- `monokai` - Dark theme (recommended for dark code blocks)
- `github` - Light GitHub style
- `atom-one-dark` - Atom editor dark theme
- `vs` - Visual Studio light theme
- `dracula` - Popular dark theme
